## How to run the project locally
1. Clone the repository: `git clone <link>`
2. Open the project folder in **Visual Studio Code**.
3. For correct display, use the [Live Server] extension (https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).
4. Click the **‘Go Live’** button in the lower right corner of the editor.
